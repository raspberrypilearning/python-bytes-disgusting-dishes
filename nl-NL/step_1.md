<h2 class="c-project-heading--task">Print een weerzinwekkende welkomstboodschap</h2>

\--- task ---

Gebruik <code>print()</code> om een welkomstbericht voor je walgelijke receptenkaart weer te geven.

\--- /task ---

<h2 class="c-project-heading--explainer">Schets de situatie</h2>

Je bent zojuist aangenomen als drukker bij een wel heel bijzonder restaurant.  
Wat is jouw baan? Om afschuwelijke receptkaarten af te drukken voor hun nieuwste menu!

In Python wordt tekst een <strong>tekenreeks</strong>genoemd. Je kunt Python een tekenreeks op het scherm laten weergeven met <code>print()</code>.  
Elke keer dat je <code>print()</code>gebruikt, begint Python een nieuwe regel in de uitvoer.

Begin met het afdrukken van een welkomstbericht voor je walgelijke gerecht.
